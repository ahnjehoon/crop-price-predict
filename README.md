##### 새롭게 만들 컬럼 목록

- 가격(sbidpric) / 거래단량(delngprut) = Kg당 가격(kgperprice)

  거래 단량은 단위당 몇 Kg인지 나타내는 단위이며

  가격은 거래단량으로 얼마가 팔렸는지 나타낸다

  그래서 Kg당 가격을 구하고 싶으면 가격 / 거래단량

  거래 단량은 단위당 몇 Kg인지 나타내는 단위이며

  가격은 거래단량으로 얼마가 팔렸는지 나타낸다

  그래서 Kg당 가격을 구하고 싶으면 가격 / 거래단량

- 총거래가격(totalprice)

  거래(delngQy)량 x 가격 하면 거래 가격이 나옴

- 총량(totalweight)

  거래단량 x 거래량
